Average best and worst case of an algorithim

Example 1 : Simple function with same order of growth for every input
Program : 
    int getSum(int arr[], int n){
        int sum = 0
        for(int i=0;i<n;i++){
            sum = sum+arr[i]
        }
        return sum
    }

The best , average and worst case of this algorithim is N
